Hướng Dẫn Xây Dựng Ứng dụng Todo List với HTML, CSS và JavaScript

Hôm nay, ngày 21/05/2025, chúng ta sẽ cùng tạo một ứng dụng Todo List cơ bản. Ứng dụng này sẽ có giao diện đơn giản, cho phép người dùng thêm công việc, hiển thị danh sách công việc và xóa công việc khi cần. Hãy làm theo từng bước dưới đây.

Bước 1: Tạo Cấu Trúc HTML Cơ Bản

Chúng ta bắt đầu bằng cách tạo cấu trúc HTML. File HTML sẽ bao gồm một tiêu đề, một ô nhập liệu để thêm công việc, một nút "Thêm", và một danh sách để hiển thị các công việc. 

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ứng Dụng Todo List</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="container">
        <h1>Todo List</h1>
        <div class="input-section">
            <input type="text" id="todoInput" placeholder="Nhập công việc...">
            <button onclick="addTodo()">Thêm</button>
        </div>
        <ul id="todoList"></ul>
    </div>
    <script src="script.js"></script>
</body>
</html>

Giải thích

  • <!DOCTYPE html>khai báo đây là tài liệu HTML5.
  • <input id="todoInput"> là ô nhập liệu để người dùng nhập công việc.
  • <button onclick="addTodo()"> gọi hàmaddTodo() khi người dùng nhấn nút "Thêm".
  • <ul id="todoList"> à danh sách không có thứ tự, nơi các công việc sẽ được hiển thị.
  • File CSS (styles.css) và JavaScript (script.js) được liên kết để định dạng và xử lý logic.

Bước 2: Định Dạng Giao Diện với CSS

Tiếp theo, chúng ta tạo file styles.css để định dạng giao diện. Mục tiêu là làm cho ứng dụng trông gọn gàng và dễ sử dụng.

  • .input-section sử dụng flex để sắp xếp ô nhập liệu và nút "Thêm" trên cùng một hàng.
  • Các phần tử <li> trong danh sách được định dạng để hiển thị công việc và nút xóa, với màu sắc và hiệu ứng hover cho nút.

Bước 3: Xử Lý Logic với JavaScript

Bây giờ, chúng ta viết mã JavaScript trong file script.js để xử lý các chức năng: thêm công việc, hiển thị danh sách, và xóa công việc. 

// Lấy các phần tử từ DOM
const todoInput = document.getElementById('todoInput');
const todoList = document.getElementById('todoList');

// Hàm thêm công việc
function addTodo() {
    const todoText = todoInput.value.trim();
    
    if (todoText === '') {
        alert('Vui lòng nhập công việc!');
        return;
    }

    // Tạo phần tử li mới
    const li = document.createElement('li');
    li.innerHTML = `
        ${todoText}
        <button onclick="deleteTodo(this)">Xóa</button>
    `;
    
    // Thêm li vào danh sách
    todoList.appendChild(li);
    
    // Xóa nội dung ô nhập liệu
    todoInput.value = '';
}

// Hàm xóa công việc
function deleteTodo(button) {
    button.parentElement.remove();
}

// Thêm sự kiện Enter để thêm công việc
todoInput.addEventListener('keypress', function(event) {
    if (event.key === 'Enter') {
        addTodo();
    }
});

Giải thích

  • todoInput và todoList được lấy từ DOM để thao tác.
  • Hàm addTodo(): Kiểm tra nếu ô nhập liệu trống thì hiển thị cảnh báo. Nếu không, tạo một
  • mới chứa nội dung công việc và nút "Xóa", sau đó thêm vào danh sách và xóa nội dung ô nhập liệu.
  • Hàm deleteTodo(): Xóa phần tử <li> chứa nút "Xóa" được nhấn, bằng cách gọi remove() trên phần tử cha (parentElement).
  • Sự kiện keypress cho phép người dùng nhấn phím Enter để thêm công việc, thay vì chỉ nhấn nút "Thêm".

Bước 4: Kiểm Tra và Cải Thiện

Sau khi hoàn thành, hãy mở file HTML trong trình duyệt. Bạn sẽ thấy một giao diện đơn giản với ô nhập liệu, nút "Thêm", và danh sách công việc. Thử nhập một công việc, nhấn "Thêm" hoặc Enter, và kiểm tra xem công việc có hiển thị không. Sau đó, nhấn nút "Xóa" để xóa công việc.

Nếu muốn cải thiện, bạn có thể:

  • Thêm tính năng chỉnh sửa công việc.
  • Lưu danh sách vào localStorage để không mất dữ liệu khi tải lại trang.
  • Cải thiện giao diện với thêm màu sắc hoặc hiệu ứng.

Ứng dụng Todo List này là một dự án cơ bản nhưng rất hữu ích để học HTML, CSS và JavaScript. Bạn có thể mở rộng nó bằng cách thêm các tính năng như đánh dấu hoàn thành, lọc công việc, hoặc tạo giao diện responsive. Chúc bạn thành công!
